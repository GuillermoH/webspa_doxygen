getEntityPropertySchema ( Table  $model)

Returns an entity property "schema".

The schema is an associative array, using the property names as keys, and information about the property as the value.

The value part consists of at least two keys:

  • kind: The kind of property, either column, which indicates that the property stems from a database column, or association, which identifies a property that is generated for an associated table.
  • type: The type of the property value. For the column kind this is the database type associated with the column, and for the association type it's the FQN of the entity class for the associated table.

For association properties an additional key will be available

  • association: Holds an instance of the corresponding association class.
Parameters
\Cake\ORM\Table$modelThe model to introspect.
Returns
array The property schema
